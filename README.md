# TLArray パッケージ (v0.2)

## 概要

TeX on LaTeX 上で配列を扱うための基本的なマクロを提供します．

## 動作環境

LaTeX2e 上で動作します．

## 使い方

当面は doc/simple-manual-ja.pdf を参照してください．

## 更新履歴

* 2015-11-16 v0.1 公開
	* 最低限の機能を実装
* 2015-11-16 v0.2 公開
	* 出力と取得の概念を明確に分離し，関連する命令の仕様を変更
	* `\InsertArray` の仕様を変更
	* 配列に対する基本的な操作を行う命令を複数追加
	* 整数配列に対する操作を行う命令を複数追加
	* 繰り返し操作を行うための機能を追加
	* 配列を破棄する機能を追加
